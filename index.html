Configuración Impresora

Información Requerida Para configuración
Configuración Credenciales Administrador

Configuración credenciales clave administrador impresora.

Configuración Basica Información

Configuración de información del dispositivo campos "Alias/Nombre Impresora", "Ubicación del dispositivo", "Número de activo", "Nombre de la empresa" y "Persona de contacto".

Configuración Densidad De Tóner

Configuración de densidad de tóner para la impresoras de B/N.

%
Configuración Servidor Correo SMTP

Configuración E-mail activa "Escaneo a correo electronico", añade "Nombre del servidor = mailer.tecnoquimicas.com" y configuración de campos de direcciones y mensaje como "Remitente","Dirección predeterminada", "Nombre predeterminado" y "Asunto".

Habilitar Contactos

Configuración de contactos de red, activa "Busqueda rapida", "Habilitar contactos de red (Utilziando servidor LDAP)", "Dirección de servidor LDAP", Requisitos de autenticación del servidor como "Nombre de usuario", "Contraseña", "Ruta para iniciar la búsqueda (BaseDN, Raíz de búsqueda)", "Nombre del destinatario", "Dirección correo electrónico destinatario" y "Numero del fax del destinatario".

Configuración Identificación De La Red

Configuración de red TCP/IP, como "Nombre de host","Nombre de dominio(IPv4/IPv6)", DNS(IPv4) "Primario" y "Secundario".

Configuración De Red SNMP

Configuración de red SNMP habilita "Acceso SNMPv1/v2 de lectura/escritura", define la comunidad para campos "Definir nombre de comunidad", "Confirmar nombre comunidad escritura","Obtener nombre de comunidad", "Confirmar nombre comunidad lectura" y activa "Deshabilitar nombre de comunidad de lectura predeterminado de SNMPv1/v2 "public"".

Configuración De Comunicación Segura

Configuración de comunicación segura protocolo SSL/TLS, solo activa "TLS 1.2", desactiva "TLS 1.1" y "TLS 1.0"
